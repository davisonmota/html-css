Conteúdos do código marcado com a tag CODE.

Conteúdos do código sem a marcação com a tag CODE.
if (treu){ alert('Olá, Mundo!') }
Agora o conteúdos do código está marcado com a tag PRE e CODE. Dessa maneira o código fica com a identação que você colocar!
